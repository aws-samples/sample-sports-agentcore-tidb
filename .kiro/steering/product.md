# Sports Agent on AWS Bedrock AgentCore

An Sports analytics agent deployed on AWS Bedrock AgentCore Runtime. Provides intelligent sports analysis for sports playoff matchups, player statistics, injuries, and predictions.

## Core Capabilities

- RAG-powered search over sports knowledge base (TiDB vector database)
- Live player stats and matchup data from ESPN APIs
- Wikipedia integration for historical context
- Conversation memory via AgentCore Memory (summarization, semantic, user preferences)
- Streaming and non-streaming response modes

## Target Users

Sports analysts and fans seeking data-driven sports insights, particularly around playoff predictions, team comparisons, and injury impact analysis.

## Current Focus

2025-26 sports Playoffs season data and analysis.
